Data processing with praw and django

This web app uses praw library to fetch posts from reddit api and then add the posts to the django's ORM database. NLKT library is used to perform NLP tasks in the post itself and the comments.
NLTK (Natural Language ToolKit) is used to parse a corpus which is raw text into useful information that can be used later on for complex tasks such as:
  • Sentiment Analysis
  • Entity named recognition
  • Machine Translation
  • Autocompletion
  • Chatbots
  • and even more...
The basic parsing techniques wich will be performed in the posts are:
  • Word Tokenization: split a sentece into words
  • Bigrams: a set of two word tokens (ex. ("python", "rules"))
  • Stemming: normalize a word into its root form (being the root not necessarily a word)
  • Lemmatization: normalize the worls in its root form (beign the root an actual word)
  • Stop words: words that are commonly used to connect two words but doesn't give information
    • Pronouns
    • Articles
    • Prepositions
    • ...etc
